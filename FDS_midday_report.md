# 午间(11:00–14:00)误差专项诊断报告

> 诊断对象：山东省日前(D+1)直调负荷预测系统 **v6**（官方验证 MAE = 1445.62 MW，R² = 0.9292）
> 诊断窗口：官方验证窗 2026/03/01 00:00 ~ 2026/06/15 23:45（10272 点），其中午间 11:00–14:00 共 1284 点
> 跨年参照：训练期 3 折 walk-forward OOF（2025 春 / 2025 秋 / 2026 冬，共 22947 点）
> 数据来源：`FDS/output/diag_val.csv`（v6 模型无泄露预测）、`FDS/output/diag_oof.csv`（训练期 OOF）
> 诊断脚本：`exp_midday_diagnosis.py`（只读诊断，产物在 `FDS_midday/`：4 张图 + 14 张表）
>
> **合规声明**：本报告不修改任何生产代码 / v6 模型结构 / 训练流程 / 数据泄露约束；不新增特征；不调整
> threshold_corr / drift_corr / MOS 任何参数；不使用官方验证窗口重新调参。所有分析仅基于现有预测
> 结果、真实值与已有特征。真实负荷仅作评估目标（eval-only，#1）。文末候选方向**仅为诊断后候选，不代表
> 立即实施**；任何候选须经理论分析 + 数据泄露检查 + walk-forward 跨折验证 + 独立实验确认稳定收益后方可考虑入生产。
>
> 误差符号约定：`error = pred − actual`（正=高估，负=低估）；`ext_error = pred_load − actual`（外部预测自身误差）。

## 核心结论（TL;DR）

1. **午间问题真实且严重**：午间 MAE = **2835.0 MW**，是全天 1445.6 的 **196%**，是所有时段中最差的；
   午间 Bias = **−371.8 MW**（系统性低估），而全天 Bias 仅 −17.4（近乎无偏）。
2. **误差主体来自外部预测**：午间外部预测自身 MAE = 3449，模型仅将其降至 2835（改善 614，**18%**），
   即 **82% 的午间误差以现有输入不可消除**。最差 20 个午间点中 **19 个集中在 2 个常规工作日**
   （2026-04-13、2026-03-03），这两天外部预测偏差均达 ±13000 MW 量级。
3. **现有特征对午间残差信息已耗尽**：用现有全部特征拟合午间残差，时序 7:3 切分 test R² = **−0.030**
   （劣于恒定预测）。残差虽非白噪声（ACF 强自相关），但其结构**不在现有特征中**，而在未观测变量
   （实时光伏出力、云团移动、外部预测自身误差模式）。
4. **午间偏置跨年不稳定**：同季节跨年（2025 春 OOF vs 2026 春 val）午间 model_Bias 由 **+1971 翻转为
   −572**，11 时由 +2202 翻转为 −1157。任何在 2025 上调出的午间偏置校正都会在 2026 上反向。
5. **建议**：午间误差主要来自**外部预测误差 + 不可观测新能源波动 + 跨年不稳定偏置**，现有特征信息已
   耗尽。建议**停止以现有特征继续优化午间模型**，转向寻找新输入数据源（实时光伏/新能源出力、更准 NWP、
   外部预测误差实时反馈）。CS-M1/M2 候选须 walk-forward 跨年验证，但据跨年翻转证据与 CS1/CS2/CS3 先例，
   迁移概率低。

## 第一部分：午间问题是否存在 —— 存在，且为全天最差时段

按时段切分（`FDS_midday/tables/01_midday_vs_bands.csv`，图 `02_hour_profile.png`）：

→ 午间低估(−372) → 下午高估(+552)，午间处于低估侧的低谷。**午间显著劣化，问题真实存在。**

## 第三部分：午间预测曲线拟合分析（图 `03_midday_fit.png`）

- 午间整体 Bias = −371.8（**低估**）；
- **高负荷（actual P90+）点 Bias = −4316.0**（严重低估）；低负荷（actual P10−）点 Bias = +1122.0（高估）；
- pred-actual 相关 = 0.939（高），pred std / actual std = **0.963**（略 <1，波动略不足但非主因）。

**判断**：形态为 **D（高负荷区域压缩）** + 轻度波动不足。高实际负荷点被严重低估、低负荷点被高估，
呈典型的**向均值回归/压缩**。但结合第六、七部分，这种压缩的根因是**外部预测在高实际负荷点本身就漏峰**
（pred_load 不高），而非模型主动收缩——模型拿到的输入 pred_load 已缺失峰值信号（详见第六、七部分及
CS2 诊断：峰值收缩 = ext_error 不可学）。

## 第八部分：残差可学习性（图 `08_acf_pacf.csv`，`08_acf_pacf.png`）

- 全程残差 ACF：lag1 = **0.976**，lag4 = 0.887，lag96 = 0.304，lag192 = 0.174；
- 前 50 阶 ACF 超 ±1.96/√N 的个数 = **50/50**（白噪声期望 ~5%）→ **残差非白噪声，含强时间结构**；
- 午间日均残差 ACF(lag1天) = 0.322，ACF(lag7天) = 0.085 → 午间偏置有中度日间持续性；
- **信息耗尽检验**（午间残差 ~ 现有全部特征，时序 7:3 切分）：test **R² = −0.030**（劣于恒定预测），
  Ridge MAE 2739 > 恒定 MAE 2592。

**判断**：残差**有时间规律**（强短程自相关 = 外部预测一旦系统性偏一段就持续数小时；中度日间持续性），
**但现有特征无法捕获**（R² ≈ 0）。这说明规律存在于**未观测变量**——实时光伏出力、云团实时移动、外部预测
自身的误差模式——而非当前特征集。**当前模型已充分利用现有信息。**

## 综合结论

### 1. 午间问题是否真实存在
**存在。** 午间 MAE 2835.0 = 全天 1445.6 的 196%，为六时段最差；午间 Bias −371.8（系统性低估）。
问题可进一步拆为：11:00–11:45 系统低估（slot 44–47，Bias −850~−1420）+ 12:00–13:59 近无偏高方差
（MAE 2800+，std 3700+）。

### 2. 主要误差来源排序
1. **外部预测负荷误差（第一大）**：午间外部 MAE 3449，模型仅降至 2835（改善 18%），82% 误差以现有输入
   不可消除；Top19/20 失败点集中在 2 个外部预测崩盘日（±13000 MW）；P90–95 峰值漏峰 ext_Bias −3485。
2. **午间光伏/新能源波动不可预测（第二大）**：多云午间（clearness 0.2–0.5）MAE 3818、std 4827 为各档
   最高，误差主体是方差（日前不可预测的光伏波动）；残差非白噪声但 R²≈0 → 结构在未观测新能源变量。
3. **高负荷外部漏峰（第三大）**：高 actual P90+ Bias −4316；P90–95 pred_load 外部漏峰 −3485，模型无信号
   定位（=ext_error，与 CS2/FDS 一致）。
4. **11:00 爬坡定时偏差（第四大）**：slot 44–47 系统低估，看似可校正，但跨年符号翻转（不可迁移）。
5. **天气条件失配（第五大）**：threshold_corr 方向有效但多云档仅动偏置不动方差，且校正幅度跨年漂移。
6. **关键变量缺失（贯穿性）**：现有特征对午间残差 R²=−0.03，信息耗尽；缺失实时光伏/新能源出力、实时
   云图、外部预测误差反馈。

### 3. 当前模型已经解决的问题
- 整体 MAE 1445.6（午间外部 3449 → 模型 2835，改善 614）；
- clearness 0.5–0.8 完全校正（Bias −23）；clear≥0.8 部分校正（ext +577 → −345）；
- 高 pred_load P95–99 / >P99 外部高估处模型校正良好（+577 / +241）；
- threshold_corr / drift_corr / MOS 方向正确并已纳入 v6；
- 夜间/晚高峰 MAE 低（622 / 809），非午间时段拟合良好。

### 4. 当前模型未解决的问题
- 11:00–11:45 爬坡系统低估（−1139，3 月最差 −1153）；
- 多云午间高方差（MAE 3818，std 4827）；
- P90–95 外部漏峰（ext −3485，模型无法定位）；
- 极端日外部崩盘（2026-04-13 / 2026-03-03，±13000 MW）；
- 跨年偏置不稳定（2025春 +1971 → 2026春 −572，符号翻转）。

### 5. 是否存在可靠改进空间
**以现有特征集，可靠改进空间接近 0。** 证据：(a) 信息耗尽 R²=−0.030；(b) 跨年偏置符号翻转；
(c) 误差主体为外部预测 + 新能源波动（不可从 pred_load 定位）；(d) CS1/CS2/CS3 三个既有候选经
walk-forward 跨年验证均已判不可迁移。继续在现有特征/模型结构上调参，单窗口或可降 MAE，但**不可跨年迁移**
（过拟合验证窗）。

### 6. 候选改进方向（仅为诊断后候选，不代表立即实施）

> 以下候选须依次通过：理论分析 → 数据泄露检查 → walk-forward 跨折验证 → 独立实验，确认稳定收益后方可考虑入生产。

**CS-M1：11:00 爬坡 slot 偏置再校正**
- 理论依据：slot 44–47 在 val 窗内低估 −850~−1420 稳定，hour_bias 已部分处理但残差仍 −1139。
- 预期收益：若跨年稳定，或可降午间 MAE ~50–100 MW。
- 风险：**跨年符号翻转**（2025春 +2202 → 2026春 −1157），极可能不迁移（同 CS3 模式）。
- 验证方式：walk-forward 3 折逐 slot 估计 11 时 shift，检验跨折方向一致性；不得用官方验证窗选参。

**CS-M2：多云午间（clearness 0.2–0.5）偏置再校正**
- 理论依据：现有 threshold_corr 仅校 +594，残 −480。
- 预期收益：偏置部分或可再降 ~100 MW。
- 风险：该档误差**方差主导**（std 4827），偏置可调空间小；CS3 已证多云校正跨年幅度漂移、不可迁移。
- 验证方式：同 CS3 walk-forward 跨年稳定性验证。

**CS-M3：引入新输入数据源（唯一具突破潜力的方向）**
- 理论依据：R²≈0 证明现有特征信息耗尽，残差结构在未观测变量（实时光伏/新能源出力、实时云图、外部预测
  误差实时反馈）。
- 预期收益：才可能突破 2835 下限，从根本上补足外部预测漏峰与光伏波动盲区。
- 风险：数据可得性、引入泄露风险、部署管线成本；须严格做泄露检查（新数据不得含 actual 衍生量）。
- 验证方式：取得数据后构造增量特征，walk-forward 跨年验证增量收益。

## 最终原则建议

诊断证明：**午间误差主要来自外部预测误差 + 不可观测新能源波动 + 跨年不稳定偏置**，现有特征信息已耗尽
（R²=−0.030），跨年偏置符号翻转。依 /goal 最终原则，**明确建议：停止以现有特征继续优化午间模型**，
转向**寻找新输入数据源**（实时光伏/新能源出力、更准 NWP、外部预测误差实时反馈）。CS-M1/M2 须经
walk-forward 跨年验证，但据跨年翻转证据与 CS1/CS2/CS3 先例，迁移概率低；CS-M3 是唯一可能突破当前
信息下限的方向，应作为下一步重点评估对象（前提是数据可得且通过泄露检查）。
