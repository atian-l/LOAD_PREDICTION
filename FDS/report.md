# 预测诊断报告 (Forecast Diagnosis) - v6 负荷预测系统

**诊断对象**：生产系统 v6（40 成员 LightGBM median 集成 + Ridge MOS + 96-dim hour_bias + drift_corr + threshold_corr + 负荷加权）
**验证窗**：2026/03/01 00:00 – 2026/06/15 23:45（10272 点）
**整体指标**：MAE = **1445.62 MW**，RMSE = 2274.42，Bias = −17.38 MW（t=−0.8, p=0.44，**无系统偏移**），MAPE = 2.43%
**对照**：外部预测 pred_load 自身 MAE = 1660.43 MW；模型校正净改善 214.81 MW（校正有效率 12.6%）；训练期 OOF（未校正）MAE = 1724.92

> 本报告为**纯诊断**：基于 v6 已有预测结果做多维误差分析，未修改任何生产代码/模型/训练/特征，未进行任何降 MAE 实验。全部图表见 `FDS/output/figures/`，数据表见 `FDS/output/tables/`。

## 一、误差分布（图 01）

- **无系统偏移**：Bias = −17.4 MW，t 检验 p=0.44，不显著。模型整体无高估/低估偏移。
- **重尾、尖峰**：偏度 +0.32，峰度 5.38（远高于正态的 3）。QQ 图两端明显偏离正态。
- **长尾主导 MAE**：|error|<500 MW 的点占 34.3% 但仅贡献 82 MW；|error|>4k MW 的点仅占 8.5% 却贡献 **516 MW**（占 MAE 的 36%）。少数极端点决定整体水平。
- 高估 46% / 低估 54%，基本对称。

## 二、时间规律（图 02，表 02_*）

- **季节/月份**：6 月（夏）最差 MAE=1740、Bias=+121；4 月最佳 MAE=1260。春季 MAE=1398、夏季 1740。
- **周末恶化**：周末 MAE=1533（Bias+211）> 工作日 1410（Bias−111）。周末负荷模式预测更差。
- **节假日反而更准**：节假日 MAE=1282 < 非节假日 1459（Bias −139）。节假日已被 `is_holiday` 特征较好捕获。
- 小时 MAE 有明显日内规律（午间光伏时段高），已被 96-dim hour_bias 大部吸收但残留。

## 三、负荷区间（图 03，表 03_*）— 关键发现

- **U 型**：P0–10（MAE 1888）与 P90–100（MAE 1912）最差，中段（P30–40）最佳（948）。
- **峰值严重低估**：实际≥P95（75543 MW）的点 MAE=2700、**Bias=−2016 MW**（系统性低估 2 GW）。
- **谷值轻微高估**：实际≤P5（46824 MW）Bias=+429。
- ⚠️ **峰值低估是模型收缩，非数据外推**：验证期峰值 max=87885 MW、P99=80105，均**低于**训练期 max=111101、P99=92445（验证峰值超训练 max 占比=0%）。模型见过更高峰值仍低估 → 属 LightGBM median/quantile 回归向均值收缩的**模型能力问题**，非数据不可得。

## 四、天气条件（图 04，表 04_*）

- **clearness**：0.2–0.4（多云）MAE=2124、>0.8（晴）MAE=2343 最差；<0.2（夜）718 最佳。午间光伏相关天气是误差重灾区。
- **温度**：低温（<0）与高温（>28）误差上升。
- **降水**：有雨 MAE=1744 高于无雨。
- **风**：大风（>8）反而 MAE=1028 较低（风电可预测性较好）。

## 五、二维热力图（图 05）— 系统性失配区

- **最大失配**：温度 0–8°C × 小时 10–14 = Bias **−5647 ~ −4181 MW**（寒冷午间严重低估）。该区已被 `temp<8` threshold_corr 部分校正，但残留仍极大。
- 晴午（clearness>0.8 @11–14）低估、高温时段高估的二维结构清晰可见。

## 六、残差自相关（图 06，表 06）

- ACF：lag1=**0.976**，lag4=0.887，lag96=0.304，lag192=0.174；200/200 阶显著。
- PACF：lag1=0.988，lag4≈0（之后近 0）→ 残差近似一阶马尔可夫。
- **解读**：lag1 极高主要源于**负荷目标本身平滑**（相邻 15min 负荷几乎相同，误差亦然），属目标性质，非"可操作未学规律"。lag96=0.304 的日级相关经 PACF 证实多由短 lag 传递（PACF@96≈0），直接的日级结构已被吸收。

## 七、连续误差（图 07，表 07）— 日级漂移

- 连续低估 run 更长（均 11.1 vs 高估 9.5；最长 156 步≈39h）；≥1 天的 run：低估 4 段、高估 2 段。
- **逐日 Bias 波动剧烈**：均值 −17、std=1374、范围 [−3762, +5880] MW。
- **日级方差占总误差方差 36.5%** — 这是外部预测逐日特异性误差的直接体现，是不可迁移的主导成分。

## 八、爬坡（图 08，表 08）

- 大爬坡（|Δ|>800 MW/15min）占 39% 点，MAE=1695（高于全体 1446）。
- **方向预测成功**：大爬坡（>1500）>98%，小爬坡（−100~100）仅 60%（小爬坡方向本就噪声大）。
- 大上行爬坡 Bias=+311（过估上行幅度），大下行 Bias=−194。模型对快速变化的**幅度**估计不足，但方向判断可靠。

## 十、误差来源拆解（图 10，表 10）— 核心结论

方差分解（sequential OLS R² 增量）：
| 因子组 | 边际方差解释% |
|---|---|
| 外部预测误差 ext_error | **83.7%** |
| 时间规律(时/周/月) | 6.2% |
| 天气(温/晴/辐) | 2.0% |
| 节假日与周末 | 0.1% |
| 负荷水平(pl十分位) | 0.6% |
| 随机残差 | 7.5% |

- **corr(e, ext_error)=0.915**，**corr(e, model_corr)=−0.073**：最终误差与外部预测误差高度相关，而模型校正量与最终误差几乎不相关 → 模型校正方向与"真实误差方向"对齐度低（因真实误差方向不可从特征获知）。
- 模型仅消除外部误差幅度的 12.6%；**84% 的残差方差由外部预测的特异性误差驱动，predict-time 不可得 → 不可学习**。

## 十一、特征贡献（图 11，表 11_*）

- **pred_load 绝对主导**：SHAP 均值 2624（次位 pred_load_lag_96 仅 378），置换重要性 +7342 MW（次位仅 +16）。模型本质是"精炼的外部预测"。
- **有效特征**：pred_load、pred_load_lag_96、pred_load_diff_96、irrad_range、mod_cos、hour_cos、pl_x_hour、pl_x_irrad_anom、pred_load_lag_672。
- **⚠️ 负置换重要性特征（置换后 MAE 反降 = 过拟合噪声）**：
  - pred_load_vs_mean_672 (−6.3)、pred_load_roll_mean_96 (−2.4)、pred_load_roll_mean_672 (−2.1)、pred_load_lag_192 (−1.4)、pl_x_temp (−1.2)、dayofyear (−0.6)。
  - 这些特征对验证集**有害无益**，是诊断发现的明确冗余/过拟合特征。

## 十二、拟合可视化（图 12）

- 预测 vs 实际散点在峰值区（actual>75k）系统性低于 y=x 线 → 峰值低估可视化确认。
- **最差日 2026-05-20 MAE=5880**（日级特异性误差极端例）；**最佳日 2026-04-01 MAE=322**。日间表现差异 18 倍。

|---|---|---|
| 时/周/月 | 0.062 | 0.062 |
| 天气(温/晴/辐) | 0.027 | 0.088 |
| 节假日/周末 | 0.008 | 0.093 |

| pl_weather_residual | 0.009 | 0.214 |

| 随机+不可学习 | — | **0.785** |

- 结构特征总可解释 R² = 21.5%（其中负荷水平边际最大 9.3%，即峰值收缩信号）；扣除与 ext_error 重叠后，**唯一可迁移结构 ≈ 9%**。
- **逐日 oracle**：若已知当日均值偏置，MAE 1445 → **1284**（143 MW 日级上限），但 exp70/73 已证明 Ridge 与 GBM 逐日模型最优缩放均 s=0.0 → **不可迁移**。

# 最终诊断结论

### 1. 当前模型最主要的误差来源排序
1. **外部预测(pred_load)的特异性误差** — 占残差方差 83.7%，日级 36.5%。predict-time 不可得，**不可学习**。首要来源。
2. **纯随机噪声** — 7.5%。
3. **峰值系统性低估（模型收缩）** — 负荷水平边际 9.3%；P95+ 点 Bias=−2016 MW。属模型能力（非数据外推）。
4. **天气场景残留** — 雨天 48.9% MAE 贡献、多云午 MAE 3854、晴午 2410；threshold_corr 已部分校正但残留显著。
5. **时间规律残留** — 6.2%，已被 hour_bias 大部吸收。
6. **爬坡幅度** — 大爬坡 MAE 1695（方向准、幅度欠）。

### 2. 当前模型已经学到的规律
- pred_load 主导（模型即精炼外部预测）；日历周期（hour/mod/doy）；天气-负荷非线性（MOS+hdd/cdd）；pl_weather_residual 方向信号（drift_corr）；晴午/雨天/低温/多云场景偏置（threshold_corr）；96-slot 时段偏置（hour_bias）；节假日模式（holiday MAE 反而更低）。

### 3. 当前模型尚未学到的规律
- **峰值低估 2 GW**（模型收缩，非外推）；**多云午间仍低估**（MAE 3854，校正不全）；**高温过估**（+1347 MW）；**大爬坡幅度**（方向对幅度欠）；**日级特异性偏置**（±3.8~5.9 GW 日间波动）。

### 4. 哪些规律理论上仍然可以学习
- **峰值收缩**：验证峰值在训练分布内，理论上可通过 quantile alpha 上调或峰值感知机制缓解（但 exp74 负荷特征过拟合警示风险）。
- **多云午间残留偏置**：场景已识别，可重新校准 threshold shrinkage（但阈值族已近 ~1455 天花板）。
- **负置换重要性特征剪枝**：移除 pred_load_vs_mean_672 / roll_mean_96/672 / pl_x_temp 等，可能小幅降 MAE 且降复杂度。

### 5. 哪些规律由于数据限制不可学习
- **外部预测特异性误差**（83.7%，predict-time 不可得）；**日级特异性偏置**（36.5% 方差，exp70/73 s=0.0 不可迁移）；**纯随机**（7.5%）；**晨间起报精度衰减**（val 用 20:00 起报代理 9AM 晨间起报，长 lead 更不准，val 为部署下界）。

### 6. 当前模型是否已接近当前数据的信息上限
**是，已接近。** 残差 78.5% 不可学习（外部特异性 + 随机）；可迁移结构 ~9% 已被 v6 捕获；逐日 oracle 上限 1284 MW 不可达。剩余"可学"信号主要是**模型行为（峰值收缩）**而非**数据信息**，且特征工程已到上限（exp74 全部 hurt）。在当前数据与约束下，1445.62 MW 接近能力天花板；进一步下降需放宽 #1（past-actuals）或获取预报修订历史/晨间起报数据。

### 7. 哪些方向值得继续研究
- **验证体系跨窗口一致性**（方向五诊断）：验证 v6 各组件在 2025 walk-forward 折上是否一致改进，量化自适应验证风险。
- **晨间起报敏感性**（方向六诊断）：注入 lead 退化噪声估计真实部署 MAE，标注 val 为下界。
- **峰值收缩机制**（见候选 CS2）：独立评估 quantile/峰值感知是否在跨年验证中稳定。

### 8. 哪些方向应明确放弃，并说明理由
- **基于置换重要性的特征剪枝（CS1，exp84 验证）**：官方 val 上 5 个负置换重要性特征剪枝 Δ=−0.54 MW（噪声级），但 3/4 walk-forward 独立折恶化 +7~+53 MW（spring25 +52.6）。负置换重要性系窗口偶然，不可迁移；特征实为有用。**单窗口置换重要性不足以支撑剪枝决策。**
- **多云午间校正重估/调参（CS3，exp85 验证）**：多云午间偏置**方向跨年稳定（均低估）但幅度不跨年稳定**（OOF shift_est −1029~−4174 与各折实际偏差不匹配）。生产 −18.6 MW 收益源自 2026 春季单窗，不可迁移（独立 2025 秋窗 +9.0 有害）。重估/调参无稳定跨年收益。
- **新增人工特征**：exp74 验证天气导数/历史窗口/连续编码/节假日增强全部 hurt，模型已到点级特征上限。
- **残差 GBM 替代规则校正**：exp83 +446 MW 严重过拟合（规则编码可迁移结构，GBM 学不可迁移噪声）。
- **异构集成（LGB+CB+XGB）**：exp78/57 误差相关 0.986，增益 <0.4 MW。
- **非线性 MOS**：与 direct 成员功能冗余，锚点过拟合传导。
- **日级偏差模型**：exp70/73 Ridge 与 GBM 最优缩放均 s=0.0，逐日信号不可迁移。
- **追求 MAE<1300**：146 MW 缺口为不可迁移的外部特异性误差，leakage-free 不可行。

## 候选建议（诊断后发现，仅供参考）

> **以下仅为诊断后的候选建议，不代表应立即进入生产系统；任何建议均需经过充分理论分析、数据验证、无数据泄露检查以及独立实验评估后，确认具有稳定收益，方可考虑实施。**

- **CS1（特征剪枝）**：移除置换重要性为负的特征（pred_load_vs_mean_672、pred_load_roll_mean_96/672、pl_x_temp、dayofyear；注 `pred_load_lag_192` 虽置换重要性亦微负，但 CLAUDE.md 规定其为 mandatory minimum lag，**不得移除**，已排除）。诊断证据看似强（置换后 MAE 反降 1–6 MW），可能小幅降 MAE 并降低模型复杂度/过拟合。需独立验证剪枝后不损其它场景。
  - **【实验验证结果 exp84 - REJECTED】** 独立 walk-forward 稳定性验证（`exp84.py`，4 折原始集成就地重拟合 + 官方 val 全管线）：
    - Part A 官方 val 全管线：基线 1445.62（精确复现 v6）→ 剪枝 1445.08，Δ=−0.54 MW（噪声级）。
    - Part B 4 折 walk-forward 原始集成 Δ：spring25 **+52.6**、autumn25 **+7.2**、winter26 **+16.0**、official −3.0。**3/4 独立折剪枝明显有害**（最大恶化 +52.6 MW），stable=False。
    - Part C 官方 val 逐特征 LOO：vs_mean_672 −2.4 / roll_mean_96 **+2.8**（即便在 val 上也有害）/ roll_mean_672 −4.9 / pl_x_temp −0.8 / dayofyear −0.9，符号不一致，聚合 −0.54 系相互抵消。
    - **结论：CS1 放弃。** 官方 val 上置换重要性为负系窗口偶然/过拟合，**不可迁移**到独立折；这些特征实为有用，剪枝在 spring25 折恶化 52.6 MW。验证了诊断阶段"先诊断后结论"的审慎原则--单窗口置换重要性不足以支撑剪枝决策。
- **CS2（峰值收缩缓解）**：针对实际高负荷区（P75+，峰值低估 −2 GW，且非外推），评估对 residual 成员上调 quantile alpha 或加入峰值感知收缩。风险：exp74 负荷相关特征过拟合，必须跨年验证。
- **CS3（多云午间校准）**：clr∈[0.2,0.5)@11-14 残留 MAE 3854，现 threshold_corr 未充分校正；可重估该场景 OOF shift 与 shrinkage。风险：阈值族已近天花板，增益可能有限。
  - **【实验验证结果 exp85 - REJECTED】** 独立 walk-forward 跨年稳定性验证（`exp85_cs3.py`，4 折；每折训练区域 OOF 子折就地估计 shift，固定生产阈值 clearness∈[0.2,0.5)@11-14 shrinkage=1.0 不重搜，应用于 held-out val）：
    - 各折 Δ（treat−base，负=有益）：spring25 −3.7、autumn25 **+9.0**（有害）、winter26 −2.5、official **−18.6**。内 3 折 1/3 明显有害，stable=False。
    - 关键诊断--**方向稳定、幅度漂移**：4 折实际偏置 bias_val 均为负（多云午间系统性低估，物理规律跨年成立），但 OOF 估 shift_est 在 −1029~−4174 间剧烈波动且与各折实际偏差不匹配：autumn25 估 −2160 vs 实际 −575（过校正 3.8 倍 -> +9.0 有害）；winter26 估 −1029 vs 实际 −3345（欠校正）；spring25 估 −4174 vs 实际 −2305（过校正 1.8 倍）。**仅 official 折 OOF 估值 −2447 恰近实际 −2643 -> Δ=−18.6**，此幅度对齐部分偶然。
    - Sanity：official treat_MAE=1445.62（精确复现 v6），base（去多云午间）=1464.19。
    - **结论：CS3 放弃。** 多云午间偏置**方向跨年稳定但幅度不跨年稳定**，任何单一 OOF shift 估值都会在某些 held-out 期失准；生产 −18.6 MW 收益源自 2026 春季单窗且经大量 val 选择，**不可迁移**（独立 2025 秋窗 +9.0 有害）。与 CS1 对照：CS1 系"val 噪声冒充有害特征"，CS3 系"方向真信号但幅度跨年漂移"--单窗口结论均被 walk-forward 证伪。注：方向既稳定，理论上一致性收缩（shrinkage<1.0）可降低过校正风险，但择优 shrinkage 需 val 选择（违约束），不作采纳。
